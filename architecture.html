Factory Inventory Management — Architecture

Vue 3 single-page app backed by a FastAPI service with in-memory JSON mock data.

Tech Stack

Frontend

Vue 3 SPA

  • Vue 3 + Composition API
  • Vue Router 4
  • Vite 5 dev server
  • Axios HTTP client
  • i18n (en / ja) via composable

Port 3000

Backend

FastAPI Service

  • Python FastAPI
  • Pydantic models for validation
  • Uvicorn ASGI server
  • CORS open (dev only)
  • Auto-generated OpenAPI docs

Port 8001

Data

In-Memory JSON

  • Loaded at startup from server/data/*.json
  • No database — changes do not persist
  • Python filtering in request handlers
  • Suitable for demo / small datasets
